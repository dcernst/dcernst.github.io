If we think of a set as a box potentially containing some stuff, then the empty set is a box with nothing in it. One assumption we will make is that for any set \(A\text{,}\) \(A\notin A\text{.}\) The language associated to sets is specific. We will often define sets using the following notation, called set-builder notation:
\begin{equation*} \tcboxmath{S=\{x \in A\mid P(x)\}}\text{,} \end{equation*}
where \(P(x)\) is some predicate statement involving \(x\text{.}\) The first part “\(x \in A\)” denotes what type of \(x\) is being considered. The predicate to the right of the vertical bar (not to be confused with “divides” ) determines the condition(s) that each \(x\) must satisfy in order to be a member of the set. This notation is read as “The set of all \(x\) in \(A\) such that \(P(x)\text{.}\)” As an example, the set \(\{x\in \mathbb{N}\mid x \mbox{ is even and } x\geq 8\}\) describes the collection of even natural numbers that are greater than or equal to 8.
