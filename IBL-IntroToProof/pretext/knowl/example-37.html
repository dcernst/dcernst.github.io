Example 9.4.

Let \(\mathbb{R}^+\) denote the set of positive real numbers and define \(f:\mathbb{R}\to \mathbb{R}^+\) via \(f(x)=e^x\text{.}\) As you are likely familiar with, this exponential function is a bijection, and so \(\card(\mathbb{R})=\card(\mathbb{R}^+)\text{.}\) Similar to the previous example, we could also use the inverse function \(f^{-1}:\mathbb{R}^+\to \mathbb{R}\) given by \(f^{-1}(x)=\ln(x)\) to show that these two sets have the same cardinality.
