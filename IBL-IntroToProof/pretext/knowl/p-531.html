In the special case that \(\Delta=\mathbb{N}\text{,}\) we write
\begin{equation*} \bigcup_{n=1}^{\infty}A_n= \{ x \mid x \in A_n \mbox{ for some } n \in \mathbb{N}\}= A_1\cup A_2 \cup A_3 \cup \cdots \end{equation*}
and
\begin{equation*} \bigcap_{n=1}^{\infty}A_n= \{ x \mid x \in A_n \mbox{ for all } n \in \mathbb{N}\} = A_1\cap A_2 \cap A_3 \cap \cdots \end{equation*}
