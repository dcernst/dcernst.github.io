Recall from DefinitionÂ 9.38 that a set \(A\) is uncountable if \(A\) is not countable. Since all finite sets are countable, the only way a set could be uncountable is if it is infinite. It follows that a set \(A\) is uncountable if and only if there is never a bijection between \(\mathbb{N}\) and \(A\text{.}\) It is not clear that uncountable sets even exist! It turns out that uncountable sets do exist and in this section, we will discover a few of them.
