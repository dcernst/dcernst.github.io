Here is one possible approach for proving the next theorem. Make a table with column headings \(0, 1, -1, 2,-2,\ldots\) and row headings \(1,2,3,4,5,\ldots\text{.}\) If a column has heading \(m\) and a row has heading \(n\text{,}\) then the entry in the table corresponds to the fraction \(m/n\text{.}\) Find a way to zig-zag through the table making sure to hit every entry in the table (not including column and row headings) exactly once. This justifies that there is a bijection between \(\mathbb{N}\) and the entries in the table. Do you see why?
