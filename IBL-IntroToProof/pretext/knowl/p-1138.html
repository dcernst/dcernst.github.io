When \(\sim\) is an equivalence relation on a set \(A\text{,}\) it is common to write each equivalence class \(\rel(a)\) as \(\tcboxmath{[a]}\) (or sometimes \(\overline{a}\)). The element \(a\) inside the square brackets is called the representative of the equivalence class \([a]\text{.}\) Theorem 7.45 implies that an equivalence class can be represented by any element of the equivalence class. For example, in Problem 7.39, we have \([1]=[6]\) since 1 and 6 are in the same equivalence class. The collection of equivalence classes \(\Rel(\sim)\) is often denoted by \(\tcboxmath{A/\mathord\sim}\text{,}\) which is read as “\(A\) modulo \(\sim\)” or “\(A\) mod \(\sim\)”. The collection \(A/\mathord\sim\) is sometimes referred to as the quotient of \(A\) by \(\sim\).
