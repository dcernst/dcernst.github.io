Loosely speaking, a real function \(f\) is continuous at the point \(a\in\dom(f)\) if we can get \(f(x)\) arbitrarily close to \(f(a)\) by considering all \(x\in\dom(f)\) sufficiently close to \(a\text{.}\) The value \(\varepsilon\) is indicating how close to \(f(a)\) we need to be while the value \(\delta\) is providing the “window” around \(a\) needed to guarantee that all points in the window (and in the domain) yield outputs within \(\varepsilon\) of \(f(a)\text{.}\) Figure 8.99 illustrates our definition of continuity. Note that in the figure, the point \(a\) is fixed while we need to consider all \(x\in\dom(f)\) such that \(|x-a|\lt \delta\text{.}\) The dashed box in the figure has dimensions \(2\delta\) by \(2\varepsilon\) and is centered at the point \((a,f(a))\text{.}\) Intuitively, the function is continuous at \(a\) since given \(\varepsilon>0\text{,}\) we could find \(\delta >0\) so that the graph of the function never exits the top or bottom of the dashed box.
