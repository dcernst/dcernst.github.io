Definition 3.32.

Let \(\{A_{\alpha}\}_{\alpha\in\Delta}\) be a collection of sets.
  1. The union of the entire collection is defined via
    \begin{equation*} \tcboxmath{\bigcup_{\alpha\in\Delta} A_{\alpha}\coloneqq \{x\mid x\in A_{\alpha} \mbox{ for some } \alpha\in \Delta\}}\text{.} \end{equation*}
  2. The intersection of the entire collection is defined via
    \begin{equation*} \tcboxmath{\bigcap_{\alpha\in\Delta} A_{\alpha}\coloneqq \{x\mid x\in A_{\alpha} \mbox{ for all } \alpha\in \Delta\}}\text{.} \end{equation*}
