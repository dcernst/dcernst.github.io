Any set (finite or infinite) can be used as an indexing set. Often capital Greek letters are used to denote arbitrary indexing sets and small Greek letters to represent elements of these sets. If the indexing set is a subset of \(\mathbb{R}\text{,}\) then it is common to use Roman letters as individual indices. Of course, these are merely conventions, not rules.
  • If \(\Delta\) is a set and we have a collection of sets indexed by \(\Delta\text{,}\) then we may write \(\{S_{\alpha}\}_{\alpha\in \Delta}\) to refer to this collection. We read this as “the set of \(S\)-sub-alphas over alpha in Delta.”
  • If a collection of sets is indexed by \(\mathbb{N}\text{,}\) then we may write \(\{U_n\}_{n\in\mathbb{N}}\) or \(\{U_n\}_{n=1}^{\infty}\text{.}\)
  • Borrowing from this idea, a collection \(\{A_1,\ldots,A_{26}\}\) may be written as \(\{A_n\}_{n=1}^{26}\text{.}\)
