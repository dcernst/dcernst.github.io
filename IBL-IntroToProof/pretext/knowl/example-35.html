Example 9.2.

Let \(A=\{1,2,3,4,5\}\) and \(B=\{\text{ apple } ,\text{ banana } ,\text{ cherry } ,\text{ dragon fruit } ,\text{ elderberry } \}\text{.}\) The function \(f:A\to B\) given by
\begin{equation*} f=\{(1,\text{ apple } ),(2,\text{ banana } ),(3,\text{ cherry } ),(4,\text{ dragon fruit } ),(5,\text{ elderberry } )\} \end{equation*}
is a bijection from \(A\) to \(B\text{,}\) and hence \(\card(A)=\card(B)\text{.}\) Note that this is not the only bijection from \(A\) to \(B\text{.}\) In fact, there are \(5!=120\) bijections from \(A\) to \(B\text{,}\) one of which is the function \(f\) we defined above. The inverse of each bijection from \(A\) to \(B\) is a bijection from \(B\) to \(A\text{.}\) We could also use any of of these bijections to verify that \(\card(A)=\card(B)\text{.}\)
