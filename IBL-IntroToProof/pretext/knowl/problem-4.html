Problem 2.12.

Determine whether the converse of each of Corollary 2.9, Theorem 2.10, and Theorem 2.11 is true. That is, for \(a,n,m\in\mathbb{Z}\text{,}\) determine whether each of the following statements is true or false. If a statement is true, prove it. If a statement is false, provide a counterexample.
  1. If \(a\) divides \(n^2\text{,}\) then \(a\) divides \(n\text{.}\) (Converse of Corollary 2.9)
  2. If \(a\) divides \(-n\text{,}\) then \(a\) divides \(n\text{.}\) (Converse of Theorem 2.10)
  3. If \(a\) divides \(m+n\text{,}\) then \(a\) divides \(m\) and \(a\) divides \(n\text{.}\) (Converse of Theorem 2.11)
