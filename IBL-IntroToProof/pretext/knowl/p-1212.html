For \(n\in \mathbb{N}\text{,}\) let \(\tcboxmath{[a]_n}\) denote the equivalence class of \(a\) with respect to \(\equiv_n\) (see Definitions 7.20 and Definition 7.47). The equivalence class \([a]_n\) is called the congruence (or residue) class of \(a\) modulo \(n\text{.}\) The collection of all equivalence classes determined by \(\equiv_n\) is denoted \(\tcboxmath{\mathbb{Z}/n\mathbb{Z}}\text{,}\) which is read “\(\mathbb{Z}\) modulo \(n\mathbb{Z}\)”.
