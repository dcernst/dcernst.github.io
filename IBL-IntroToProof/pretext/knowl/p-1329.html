For each of the following, determine whether the description determines a well-defined function.
  1. Define \(f:\mathbb{Z}/5\mathbb{Z}\to \mathbb{N}\) via
    \begin{equation*} f([a]_5)=\begin{cases}0, \amp \mbox{if } a\mbox{ is even} \\ 1, \amp \mbox{if } a\mbox{ is odd} . \end{cases} \end{equation*}
  2. Define \(g:\mathbb{Z}/6\mathbb{Z}\to \mathbb{N}\) via
    \begin{equation*} g([a]_6)=\begin{cases}0, \amp \mbox{if } a\mbox{ is even} \\ 1, \amp \mbox{if } a\mbox{ is odd} . \end{cases} \end{equation*}
  3. Define \(m:\mathbb{Z}/8\mathbb{Z}\to \mathbb{Z}/10\mathbb{Z}\) via \(m([x]_{8})=[6x]_{10}\text{.}\)
  4. Define \(h:\mathbb{Z}/10\mathbb{Z}\to \mathbb{Z}/10\mathbb{Z}\) via \(h([x]_{10})=[6x]_{10}\text{.}\)
  5. Define \(k:\mathbb{Z}/43\mathbb{Z}\to \mathbb{Z}/43\mathbb{Z}\) via \(k([x]_{43})=[11x-5]_{43}\text{.}\)
  6. Define \(\ell:\mathbb{Z}/15\mathbb{Z}\to \mathbb{Z}/15\mathbb{Z}\) via \(\ell([x]_{15})=[5x-11]_{15}\text{.}\)
