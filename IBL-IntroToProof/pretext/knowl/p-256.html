We must take care to specify the collection of acceptable values for the free variables. Consider the sentence “For all \(x\text{,}\) \(x>0\text{.}\)” Is this sentence true or false? The answer depends on what set the universal quantifier applies to. Certainly, the sentence is false if we apply it for all \(x\in\mathbb{Z}\text{.}\) However, the sentence is true for all \(x\in\mathbb{N}\text{.}\) Context may resolve ambiguities, but otherwise, we must write clearly: “For all \(x\in\mathbb{Z}\text{,}\) \(x>0\)” or “For all \(x\in\mathbb{N}\text{,}\) \(x>0\text{.}\)” The collection of intended values for a variable is called the universe of discourse.
