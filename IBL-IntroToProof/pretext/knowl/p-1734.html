Before we get started, recall that every number in \((0,1)\) can be written in decimal form. However, there may be more than one way to write a given number in decimal form. For example, \(0.2\) equals \(0.1\overline{99}\text{.}\) A number \(0.a_1a_2a_3\ldots\) in \((0,1)\) is said to be in standard decimal form if there is no \(k\) such that for all \(i>k\text{,}\) \(a_i=9\text{.}\) That is, a number is in standard decimal form if and only if its decimal expansion does not end with a repeating sequence of 9â€™s. For example, \(0.2\) is in standard decimal form while \(0.1\overline{99}\) is not, even though both represent the same number. It turns out that every real number can be expressed uniquely in standard decimal form. We will take this fact for granted.
