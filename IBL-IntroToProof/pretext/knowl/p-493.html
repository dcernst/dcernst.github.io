The above paradox is one way of phrasing a paradox referred to as Russell’s Paradox, named after British mathematician and philosopher Bertrand Russell 1  (1872–1970). How did we get into this mess in the first place?! By assuming the existence of a set of all sets, we can produce all sorts of paradoxes. The only way to avoid these types of paradoxes is to conclude that there is no set of all sets. That is, the collection of all sets cannot be a set itself.
