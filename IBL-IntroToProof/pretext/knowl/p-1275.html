Since each element \(X\) appears exactly once as a first coordinate, \(f\) is a function with domain \(X\) and codomain \(Y\) (i.e., \(f:X\to Y\)). In this case, we see that \(\range(f)=\{1,2,4\}\text{.}\) Moreover, we can write things like \(f(a)=2\) and \(c\mapsto 4\text{,}\) and say things like “\(f\) maps \(b\) to 4” and “the image of \(d\) is 1.” The function diagram for \(f\) is depicted in Figure 8.3.
