To handle the first case, use induction together with Theorem 9.46. The second case is substantially more challenging. First, use Theorem 9.47 to construct a collection \(\{B_n\}\) of pairwise disjoint sets whose union is equal to the union of the original collection. Since each \(B_n\) is a subset of one of the sets in the original collection and each of these sets is countable, each \(B_n\) is also countable by Theorem 9.42. This implies that for each \(n\text{,}\) we can write \(B_n=\{b_{n,1}, b_{n,2},b_{n,3},\ldots\}\text{,}\) where the set may be finite or infinite. From here, we outline two different approaches for continuing. One approach is to construct a bijection from \(\mathbb{N}\) to \(\bigcup_{n=1}^{\infty}B_n\) using Figure 9.48 as inspiration. One thing you will need to address is what to do when a set in the collection \(\{B_n\}\) is finite. For the second approach, define \(f:\bigcup_{n=1}^{\infty}B_n\to \mathbb{N}\) via \(f(b_{n,m})=2^n3^m\text{,}\) verify that this function is injective, and then appeal to Theorem 9.42.
