Problem 2.86.

Determine whether each of the following statements is true or false. If the statement is true, prove it. If the statement is false, provide a counterexample.
  1. For all \(n\in\mathbb{N}\text{,}\) \(n^2\geq 5\text{.}\)
  2. There exists \(n \in \mathbb{N}\) such that \(n^2-1=0\text{.}\)
  3. There exists \(x \in \mathbb{N}\) such that for all \(y \in \mathbb{N}\text{,}\) \(y \leq x\text{.}\)
  4. For all \(x\in\mathbb{Z}\text{,}\) \(x^3\geq x\text{.}\)
  5. For all \(n\in\mathbb{Z}\text{,}\) there exists \(m\in\mathbb{Z}\) such that \(n+m=0\text{.}\)
  6. There exists integers \(a\) and \(b\) such that \(2a+7b=1\text{.}\)
  7. There do not exist integers \(m\) and \(n\) such that \(2m+4n=7\text{.}\)
  8. For all \(a, b, c\in\mathbb{Z}\text{,}\) if \(a\) divides \(bc\text{,}\) then either \(a\) divides \(b\) or \(a\) divides \(c\text{.}\)
  9. For all \(a, b\in\mathbb{Z}\text{,}\) if \(ab\) is even, then either \(a\) or \(b\) is even.
