Let \(R\) be a relation on a set \(A\text{.}\)
  1. The relation \(R\) is reflexive if for all \(a\in A\text{,}\) \(aR a\text{.}\)
  2. The relation \(R\) is symmetric if for all \(a,b\in A\text{,}\) if \(aR b\text{,}\) then \(bR a\text{.}\)
  3. The relation \(R\) is transitive if for all \(a,b,c\in A\text{,}\) if \(aR b\) and \(bR c\text{,}\) then \(aR c\text{.}\)
