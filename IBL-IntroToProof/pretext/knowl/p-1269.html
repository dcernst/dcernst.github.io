It follows immediately from the definition of function that two functions are equal if and only if they have the same domain, same codomain, and the same set of ordered pairs in the relation. That is, functions \(f\) and \(g\) are equal if and only if \(\dom(f)=\dom(g)\text{,}\) \(\codom(f)=\codom(g)\text{,}\) and \(f(x)=g(x)\) for all \(x\in X\text{.}\)
