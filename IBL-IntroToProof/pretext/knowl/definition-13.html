Definition 3.1.

A set is a collection of objects called elements. If \(A\) is a set and \(x\) is an element of \(A\text{,}\) we write \(\tcboxmath{x\in A}\text{.}\) Otherwise, we write \(\tcboxmath{x\notin A}\text{.}\) The set containing no elements is called the empty set, and is denoted by the symbol \(\tcboxmath{\emptyset}\text{.}\) Any set that contains at least one element is referred to as a nonempty set.
