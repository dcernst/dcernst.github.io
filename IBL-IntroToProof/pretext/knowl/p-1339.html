Let \(f:X\to Y\) be a function.
  1. The function \(f\) is said to be injective (or one-to-one) if for all \(y\in \range(f)\text{,}\) there is a unique \(x\in X\) such that \(y=f(x)\text{.}\)
  2. The function \(f\) is said to be surjective (or onto) if for all \(y\in Y\text{,}\) there exists \(x\in X\) such that \(y=f(x)\text{.}\)
  3. If \(f\) is both injective and surjective, we say that \(f\) is bijective.
