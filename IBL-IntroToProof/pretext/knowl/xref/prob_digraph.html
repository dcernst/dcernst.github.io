Problem 7.39.

Let \(A=\{1,2,3,4,5,6\}\) and define
\begin{equation*} R=\{(1,1),(1,6),(2,2),(2,3),(2,4),(3,3),(3,2),(3,4),(4,4),(4,2),(4,3),(5,5),(6,6),(6,1)\}\text{.} \end{equation*}
Using \(R\text{,}\) complete each of the following.
  1. Draw the digraph for \(R\text{.}\)
  2. Determine whether \(R\) is an equivalence relation on \(A\text{.}\)
  3. Find \(\Rel(R)\) by determining \(\rel(x)\) for each \(x\in A\text{.}\)
