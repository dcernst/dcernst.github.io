Example 7.45.

Let \(P\) denote the residents of a particular town and define \(\sim\) on \(P\) via \(a\sim b\) if \(a\) and \(b\) have the same last name. It is easily seen that this relation is reflexive, symmetric, and transitive, and hence \(\sim\) is an equivalence relation on \(P\text{.}\) The equivalence classes correspond to collections of individuals with the same last name. For example, Maria Garcia, Anthony Garcia, and Ariana Garcia all belong to the same equivalence class. Any Garcia can be used as a representative for the corresponding equivalence class, so we can denote it as \([\text{ Maria Garcia } ]\text{,}\) for example. The collection \(P/\mathord\sim\) consists of the various sets of people with the same last name. In particular, \([\text{ Maria Garcia } ]\in P/\mathord\sim\text{.}\)
