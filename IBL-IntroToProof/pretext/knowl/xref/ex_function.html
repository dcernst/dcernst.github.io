Example 8.2.

Let \(X=\{a,b,c,d\}\) to \(Y=\{1,2,3,4\}\) and define the relation \(f\) from \(X\) to \(Y\) via
\begin{equation*} f=\{(a,2),(b,4),(c,4),(d,1)\}\text{.} \end{equation*}
Since each element \(X\) appears exactly once as a first coordinate, \(f\) is a function with domain \(X\) and codomain \(Y\) (i.e., \(f:X\to Y\)). In this case, we see that \(\range(f)=\{1,2,4\}\text{.}\) Moreover, we can write things like \(f(a)=2\) and \(c\mapsto 4\text{,}\) and say things like “\(f\) maps \(b\) to 4” and “the image of \(d\) is 1.” The function diagram for \(f\) is depicted in Figure 8.1.
