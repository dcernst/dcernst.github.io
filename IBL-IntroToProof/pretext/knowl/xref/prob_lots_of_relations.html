Problem 7.34.

Determine whether each of the following relations is reflexive, symmetric, or transitive. In each case, you should either provide a specific counterexample or a proof.
  1. Consider the relation \(T\) described in Example 7.5.
  2. Consider the relation \(F\) described in Problem 7.21.
  3. Consider the relation \(\equiv_5\) described in Problem 7.22.
  4. Let \(P\) be the set of all people and define \(H\) via \(xHy\) if \(x\) and \(y\) have the same height.
  5. Let \(P\) be the set of all people and define \(T\) via \(xTy\) if \(x\) is taller than \(y\text{.}\)
  6. Consider the relation “divides” on \(\mathbb{N}\text{.}\)
  7. Let \(L\) be the set of lines and define \(||\) via \(l_1||l_2\) if \(l_1\) is parallel to \(l_2\text{.}\)
  8. Let \(C[0,1]\) be the set of continuous functions on \([0,1]\text{.}\) Define \(f\sim g\) if
    \begin{equation*} \int_0^1|f(x)|\ dx=\int_0^1|g(x)|\ dx\text{.} \end{equation*}
  9. Define \(R\) on \(\mathbb{N}\) via \(nR m\) if \(n+m\) is even.
  10. Define \(D\) on \(\mathbb{R}\) via \((x,y)\in D\) if \(x=2y\text{.}\)
  11. Define \(F\) on \(\mathbb{Z}\times \left(\mathbb{Z}\setminus \{0\}\right)\) via \((a,b)F(c,d)\) if \(ad=bc\text{.}\) Do you recognize this relation? Think about fractions.
  12. Define \(\sim\) on \(\mathbb{R}^2\) via \((x_1,y_1)\sim (x_2,y_2)\) if \(x_1^2+y_1^2=x_2^2+y_2^2\text{.}\)
  13. Define \(S\) on \(\mathbb{R}\) via \(xS y\) if \(\lfloor x\rfloor =\lfloor y\rfloor\text{,}\) where \(\lfloor x\rfloor\) is the greatest integer less than or equal to \(x\) (e.g., \(\lfloor \pi\rfloor=3\text{,}\) \(\lfloor -1.5\rfloor=-2\text{,}\) and \(\lfloor 4\rfloor=4\)).
  14. Define \(C\) on \(\mathbb{R}\) via \(xCy\) if \(|x-y|\lt 1\text{.}\)
