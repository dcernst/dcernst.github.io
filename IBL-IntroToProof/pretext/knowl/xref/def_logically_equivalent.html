Item e.

The proposition “\(A\) if and only if \(B\)” (alternatively, “\(A\) is necessary and sufficient for \(B\)”) is true if both \(A\) and \(B\) have the same truth value; expressed symbolically as \(\tcboxmath{A\logeq B}\) and called a biconditional proposition. If \(A\logeq B\) is true, we say that \(A\) and \(B\) are logically equivalent.
