Problem 8.98.

Suppose that \(f: \mathbb{R}\to \mathbb{R}\) is a function satisfying \(f(x+y)=f(x)+f(y)\) for all \(x,y\in\mathbb{R}\text{.}\)
  1. Prove that \(f(0)=0\text{.}\)
  2. Prove that \(f(-x)=-f(x)\) for all \(x\in\mathbb{R}\text{.}\)
  3. Prove that \(f\) is injective if and only if \(f^{-1}(\{0\})=\{0\}\text{.}\)
  4. Certainly every function given by \(f(x)=mx\) for \(m\in\mathbb{R}\) satisfies the initial hypothesis. Can you provide an example of a function that satisfies \(f(x+y)=f(x)+f(y)\) that is not of the form \(f(x)=mx\text{?}\)
