    Section 3.5 Cartesian Products of Sets

    Given a collection of sets, we can form new sets by taking unions, intersections, complements, and set differences. In this section, we introduce a type of “product” of sets. You have already encountered this concept when you learned to plot points in the plane. You also crossed paths with this notion if you have taken a course in linear algebra.

    Definition 3.44.

    For each \(n\in \mathbb{N}\text{,}\) we define an \(n\)-tuple to be an ordered list of \(n\) elements of the form \(\tcboxmath{(a_1, a_2,\ldots,a_n)}\text{.}\) We refer to \(a_i\) as the \(i\)th component (orcoordinate) of \((a_1, a_2,\ldots,a_n)\text{.}\) Two \(n\)-tuples \((a_1, a_2,\ldots,a_n)\) and \((b_1, b_2,\ldots,b_n)\) are equal if \(a_i=b_i\) for all \(1\leq i\leq n\text{.}\) A \(2\)-tuple \((a,b)\) is more commonly referred to as an ordered pair while a \(3\)-tuple \((a,b,c)\) is often called an ordered triple.
    Occasionally, other symbols are used to surround the components of an \(n\)-tuple, such as square brackets “\([\ ]\)” or angle brackets “\(\langle\ \rangle\)”. In some programming languages, curly braces “\(\{\ \}\)” are used to specify arrays. However, we avoid this convention in mathematics since curly braces are the standard notation for sets. The term “tuple” can also occur when discussing other mathematical objects, such as vectors.
    We can use the notion of \(n\)-tuples to construct new sets from existing sets.

    Definition 3.45.

    If \(A\) and \(B\) are sets, the Cartesian product (or direct product) of \(A\) and \(B\text{,}\) denoted \(A\times B\) (read as “\(A\) times \(B\)” or “\(A\) cross \(B\)”), is the set of all ordered pairs where the first component is from \(A\) and the second component is from \(B\text{.}\) In set-builder notation, we have
    \begin{equation*} \tcboxmath{A\times B\coloneqq \{(a,b)\mid a\in A, b\in B\}}\text{.} \end{equation*}
    We similarly define the Cartesian product of \(n\) sets, say \(A_1, \ldots, A_n\text{,}\) by
    \begin{equation*} \tcboxmath{\prod_{i=1}^{n} A_i\coloneqq A_1\times \cdots \times A_n\coloneqq \{(a_1,\ldots,a_n)\mid a_j\in A_j \text{ for all } 1\leq j\leq n\}}\text{,} \end{equation*}
    where \(A_i\) is referred to as the \(i\)th factor of the Cartesian product. As a special case, the set
    \begin{equation*} \underbrace{A\times \cdots \times A}_{n\text{ factors } } \end{equation*}
    is often abbreviated as \(A^n\text{.}\)
    Cartesian products are named after French philosopher and mathematician René Descartes
     8 
    en.wikipedia.org/wiki/Rene_Descartes
    (1596–1650). Cartesian products will play a prominent role in Chapter 7.

    Example 3.46.

    If \(A=\{a,b,c\}\) and \(B=\{\alpha,\beta\}\text{,}\) then
    \begin{equation*} A\times B=\{(a,\alpha), (a,\beta),(b,\alpha),(b,\beta), (c,\alpha),(c,\beta)\}\text{.} \end{equation*}

    Example 3.47.

    The standard two-dimensional plane \(\mathbb{R}^2\) and standard three space \(\mathbb{R}^{3}\) are familiar examples of Cartesian products. In particular, we have
    \begin{equation*} \mathbb{R}^2=\mathbb{R}\times \mathbb{R}=\{(x,y)\mid x,y\in \mathbb{R}\} \end{equation*}
    and
    \begin{equation*} \mathbb{R}^3=\mathbb{R}\times \mathbb{R}\times \mathbb{R}=\{(x,y,z)\mid x,y,z\in \mathbb{R}\}\text{.} \end{equation*}

    Problem 3.48.

    Consider the sets \(A\) and \(B\) from Example 3.46.
    1. Find \(B\times A\text{.}\)
    2. Find \(B\times B\text{.}\)

    Problem 3.49.

    If \(A\) and \(B\) are sets, why do you think that \(A\times B\) is referred to as a type of “product”? Think about the area model for multiplication of natural numbers.

    Problem 3.50.

    If \(A\) and \(B\) are both finite sets, then how many elements will \(A\times B\) have?

    Problem 3.51.

    Let \(A=\{1, 2, 3\}\text{,}\) \(B=\{1,2\}\text{,}\) and \(C=\{1,3\}\text{.}\) Find \(A \times B\times C\text{.}\)

    Problem 3.52.

    Let \(X=[0,1]\) and \(Y=\{1\}\text{.}\) Write each of the following using set-builder notation and then describe the set geometrically (e.g., draw a picture).
    1. \(\displaystyle X\times Y\)
    2. \(\displaystyle Y\times X\)
    3. \(\displaystyle X\times X\)
    4. \(\displaystyle Y\times Y\)

    Problem 3.53.

    If \(A\) is a set, then what is \(A\times \emptyset\) equal to?

    Problem 3.54.

    Given sets \(A\) and \(B\text{,}\) when will \(A\times B\) be equal to \(B\times A\text{?}\)

    Problem 3.55.

    Write \(\mathbb{N}\times \mathbb{R}\) using set-builder notation and then describe this set geometrically by interpreting it as a subset of \(\mathbb{R}^2\text{.}\)
    We now turn our attention to subsets of Cartesian products.

    Theorem 3.56.

    Let \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) be sets. If \(A\subseteq C\) and \(B\subseteq D\text{,}\) then \(A\times B\subseteq C\times D\text{.}\)

    Problem 3.57.

    Is it true that if \(A\times B\subseteq C\times D\text{,}\) then \(A\subseteq C\) and \(B\subseteq D\text{?}\) Do not forget to think about cases involving the empty set.

    Problem 3.58.

    Is every subset of \(C\times D\) of the form \(A\times B\text{,}\) where \(A\subseteq C\) and \(B\subseteq D\text{?}\) If so, prove it. If not, find a counterexample.

    Problem 3.59.

    If \(A\text{,}\) \(B\text{,}\) and \(C\) are nonempty sets, is \(A\times B\) a subset of \(A\times B\times C\text{?}\)

    Problem 3.60.

    Let \(A=[2,5]\text{,}\) \(B=[3,7]\text{,}\) \(C=[1,3]\text{,}\) and \(D=[2,4]\text{.}\) Compute each of the following.
    1. \(\displaystyle (A\cap B)\times (C\cap D)\)
    2. \(\displaystyle (A\times C)\cap (B\times D)\)
    3. \(\displaystyle (A\cup B)\times (C\cup D)\)
    4. \(\displaystyle (A\times C)\cup (B\times D)\)
    5. \(\displaystyle A\times (B\cap C)\)
    6. \(\displaystyle (A\times B)\cap (A\times C)\)
    7. \(\displaystyle A\times (B\cup C)\)
    8. \(\displaystyle (A\times B)\cup (A\times C)\)

    Problem 3.61.

    Let \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) be sets. Determine whether each of the following statements is true or false. If a statement is true, prove it. Otherwise, provide a counterexample.
    1. \(\displaystyle (A\cap B)\times (C\cap D)=(A\times C)\cap (B\times D)\)
    2. \(\displaystyle (A\cup B)\times (C\cup D)=(A\times C)\cup (B\times D)\)
    3. \(\displaystyle A\times (B\cap C)=(A\times B)\cap (A\times C)\)
    4. \(\displaystyle A\times (B\cup C)=(A\times B)\cup (A\times C)\)
    5. \(\displaystyle A\times (B\setminus C) = (A\times B)\setminus (A\times C)\)

    Problem 3.62.

    If \(A\) and \(B\) are sets, conjecture a way to rewrite \((A\times B)^C\) in a way that involves \(A^C\) and \(B^C\) and then prove your conjecture.
